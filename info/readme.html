Hotkeys & Play Notes

SHIFT = Hide counters

SPACE = Swap sides (solo)

ENTER = Toggle Chat

ESC = Default View

S = Toggle Score Summary

Y = Toggle Final Scoring Summary

E = Toggle Event Card Display

F = Toggle French Ministry Display

B = Toggle British Ministry Display

L = Toggle Log Display

H = To RTT Home

N = To Next RTT Game

TAB = Show/Hide Counters

A = Toggle "All Wars"

M = Scroll to Map (Top)

C = Scroll to Cards

D = Scroll to Debt

W = Scroll to War

T = Toggle Score Summary Text Mode

V = Toggle Verbosity

P = Notepad

Z = Toggle Zoom
