Imperial Struggle - Read me!

Module created by Brian Reynolds with Tor Andersson and Christophe Correia.

Rally the Troops engine by Tor Andersson.
Imperial Struggle board game designed by Jason Matthews and Ananda Gupta.
Used by permission from GMT Games.

Hotkeys

SHIFT = Hide counters
SPACE = Show/Hide Counters
ENTER = Toggle Chat
ESC = Default View
S = Toggle Score Summary
Y = Toggle Final Scoring Summary
E = Toggle Event Card Display
F = Toggle French Ministry Display
B = Toggle British Ministry Display
L = Toggle Log Display
A = Toggle "All Wars"
M = Scroll to Map (Top)
C = Scroll to Cards
D = Scroll to Debt
W = Scroll to War
T = Toggle Score Summary Text Mode
V = Toggle Verbosity
Z = Toggle Zoom
